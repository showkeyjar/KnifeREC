# 功能模块设计

## 1.数据源管理；

支持数据源：

    csv\mysql\odps

## 2.数据表管理；

表类型：

    用户\商品\商家

字段类型：

    id          编号
    text        文本
    number      数值
    category    分类
    image       图像
    datetime    时间

## 3.特征工程（高级功能）；

1.数据筛选；

    去除超过80%为空值字段；
    去除超过80%为单一值字段；

2.离群点；

    离群值使用极值替代；判断条件

2.空值填充；

    空值编码：0,-1

3.归一编码；

    onehot\labelencoder\LabelBinarizer
