# KnifeREC 推荐系统

系统预览
![view](./doc/sys_view.png)
目前支持模型：

    1.排序
    2.FM
    3.lda
    4.DeepCTR

## 安装

    pip install -r requirements.txt

## 使用

    python app.py
    
后台管理帐号：
admin@admin.com / admin
    
预测结果调用

    http://localhost:5000/predict

## Features

1.数据源管理；
2.数据预处理；
3.模型管理；
4.模型输出；
5.策略管理；
6.模型监控；
